# PhpVagas

Projeto desenvolvido como parte de uma atividade acadêmica.  

---

## Funcionalidades

- Login simples de administrador  
- CRUD de cargos e vagas  
- Exibição pública de vagas 

---

## Tecnologias utilizadas

- **PHP (CodeIgniter 4)**  
- **Bootstrap 5**  
- **MySQL**

---

## Como executar o projeto

1. **Clone o repositório**
   ```bash
   https://github.com/Gortona-dev/JobFlow.git
   ```

2. **Acesse a pasta do projeto**
   ```bash
   cd JobFlow
   ```

3. **Configure o arquivo .env**
  - copie o exemplo: .env.example

4. **Instale as dependências**
   ```bash
   composer install
   ```

5. **Inicie o servidor**
   ```bash
   php spark serve
   ```

6. **Acesse o sistema**
  - http://localhost:8080
